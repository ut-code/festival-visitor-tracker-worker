# festival-visitor-tracker-worker

To install dependencies:

```bash
bun install
```

To deploy:

```bash
bun run deploy
```

## How to use

create a POST request with body:

```json
{
  "url": "https://sample.festival.utcode.net",
  "key": "2e0c7cad39e09314a46f217c6107f96e08bd13984cd4ae4c29d96f5db440dba8"
}
```
